# v7.197 搜索质量提升实施报告

## 概述

本次更新实施了搜索质量提升的 P0 本地优化项，包括：
1. **语义去重** - 使用 Embedding 相似度替代前100字符对比
2. **查询分类器** - 自动识别查询类型
3. **动态权重** - 根据查询类型动态调整评分权重

## 新增文件

### 1. `intelligent_project_analyzer/tools/semantic_dedup.py`
语义去重模块，核心功能：
- 使用 `paraphrase-multilingual-MiniLM-L12-v2` 多语言模型
- LRU 缓存优化 Embedding 计算性能
- 可配置相似度阈值（默认 0.85）
- 自动回退到前缀去重（模型不可用时）

**主要函数**:
```python
def semantic_deduplicate(
    results: List[Dict[str, Any]],
    threshold: float = 0.85,
    content_key: str = "content",
    fallback_key: str = "snippet"
) -> Tuple[List[Dict[str, Any]], int]
```

### 2. `intelligent_project_analyzer/tools/query_classifier.py`
查询分类器模块，核心功能：
- 8 种查询类型识别：news、academic、case、concept、trend、howto、comparison、general
- 基于关键词 + 正则模式匹配
- 返回动态权重建议和优先来源

**查询类型与权重**:
| 类型 | 时效性权重 | 可信度权重 | 典型关键词 |
|------|-----------|-----------|-----------|
| news | 30% | 25% | 最新、今日、新闻 |
| academic | 5% | 45% | 论文、研究、学术 |
| case | 15% | 35% | 案例、项目、作品 |
| concept | 5% | 40% | 什么是、定义、概念 |
| trend | 25% | 30% | 趋势、流行、未来 |
| howto | 15% | 30% | 如何、教程、指南 |
| comparison | 15% | 35% | 对比、区别、vs |
