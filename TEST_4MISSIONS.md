## 常见问题排查

### 问题1: 前端显示旧格式（DeepAnalysisCard）

**原因:** 后端未输出4条使命格式

**排查步骤:**
1. 检查后端日志是否有 `[v7.302]` 标记
2. 检查 `search_question_analysis.yaml` 是否正确更新
3. 重启后端服务

**解决方案:**
```bash
# 重启后端
cd intelligent_project_analyzer
python -m uvicorn main:app --reload
```

---

### 问题2: 前端报错 "Cannot read property 'mission_1_user_problem_analysis'"

**原因:** TypeScript类型不匹配或数据结构错误

**排查步骤:**
1. 检查浏览器控制台完整错误信息
2. 检查API响应数据结构
3. 验证 `types/index.ts` 中的类型定义

**解决方案:**
```bash
# 重新编译前端
cd frontend-nextjs
npm run build
npm run dev
```

---

### 问题3: 交付物质量验证失败

**原因:** 交付物描述不符合格式要求

**排查步骤:**
1. 查看后端日志中的警告信息
2. 检查交付物是否包含"动词+对象+成果"

**解决方案:**
- 优化prompt中的交付物示例
- 调整 `_validate_deliverables()` 的验证规则
