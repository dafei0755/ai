## 🌍 核心理念：打破语言壁垒

**目标**：让中文查询也能访问全球英文网站，获取国际视野

---

## 📊 工具重新定位

### 博查 (Bocha) 的实际覆盖限制

经过官方文档调研，发现博查的关键限制：

> "搜索数据源、搜索结果符合国内安全标准，保证数据不出境"
> — 来源：[博查官方文档](https://bochaai.com/)

**这意味着**：
- 🔴 数据不出境 → 主要索引国内可访问的网站
- 🟡 内容合规 → 受国内政策限制
- 🟡 虽然声称"近百亿网页"，但国际网站覆盖受限

### Tavily 的全球能力

Tavily 作为多引擎聚合搜索：
- ✅ 真正的全球网站覆盖
- ✅ 支持多语言查询（包含中文）
- ✅ 无内容限制，可访问任何公开网站

---

## ✅ 最终方案：Tavily主导 + 智能降级

### 工具优先级

| 场景 | 优先工具 | 降级工具 | 理由 |
|------|---------|---------|------|
| **中文查询** | ✅ Tavily | 博查 | Tavily能理解中文并返回全球结果 |
| **英文查询** | ✅ Tavily | 博查 | Tavily原生全球覆盖 |
| **国内政策/报告** | 博查 | Tavily | 仅此情况优先博查 |

### 策略说明

```
任何查询 → Tavily全球搜索（优先）
              ↓ 失败时降级
            博查国内网站（备用）
              ↓ 仍失败时降级
            Serper（三级备用）
```

**关键改进**：
- 🌍 **中文查询不再受限**：如"可持续设计趋势" → Tavily查全球英文网站
- 🔍 **Tavily自动语义匹配**：中文查询 → 理解语义 → 返回相关英文结果
- 📚 **博查作为补充**：仅处理明确需要国内内容的查询

## 🔧 实施细节

### 1. 工具描述优化

**Tavily描述（完整版）**：
```
Tavily全球网络搜索 - 多语言全球覆盖，打破语言壁垒

🌍 核心能力:
- 全球网站无障碍访问：欧美、亚洲、各大洲所有主流网站
- 多语言智能理解：支持中文查询，自动匹配全球英文/多语言结果
- 语言壁垒打破：中文查询 → 全球英文网站 → 智能匹配

✅ 优先使用场景（中英文均可）:
- 任何需要全球视野的查询
- 国际设计案例："日本极简餐厅设计" / "Japanese minimalist restaurant"
- 海外技术趋势："可持续零售设计2024" / "sustainable retail 2024"

⚠️ 仅在以下情况使用博查:
- 明确只需要国内网站内容（如国内政策文件）
```

**博查描述**：
```
博查AI中文搜索引擎 - 专注中文本土内容

✅ 适用场景:
- 中文本土案例（国内品牌、中国设计师作品）
- 国内市场调研（政策、报告、行业动态）

⚠️ 不适用场景:
- 英文查询（使用Tavily）
- 国际案例（使用Tavily）
- 海外市场分析（使用Tavily）
```

### 2. 降级策略代码

**文件**：`intelligent_project_analyzer/agents/task_oriented_expert_factory.py:1856-1877`

```python
# 🌍 智能语言处理：打破语言壁垒
# 策略：中文查询 → Tavily全球搜索（自动双语）
#      英文查询 → Tavily全球搜索
# 博查仅作为中文本土内容的补充

tool_to_use = tavily_tool if tavily_tool else (bocha_tool if bocha_tool else serper_tool)

if is_chinese and tavily_tool:
    logger.info(f"🌍 [Fallback] 中文查询 '{query[:30]}...'，使用Tavily访问全球网站")
elif tavily_tool:
    logger.debug(f"🌍 [Fallback] 英文查询，使用Tavily访问全球网站")
elif bocha_tool:
    logger.warning(f"⚠️ [Fallback] Tavily不可用，降级至博查（国内网站）")
```

## 💡 实际效果示例

### 场景1：中文查询国际内容

**用户查询**："日本极简餐厅设计案例"

**处理流程**：
1. 检测到中文查询
2. ✅ 使用Tavily搜索全球网站
3. Tavily理解语义 → 匹配"Japanese minimalist restaurant design"
4. 返回全球优质英文案例

**结果**：
- ✅ 获取日本、欧美的极简餐厅设计案例
- ✅ 覆盖全球主流设计网站（Dezeen, ArchDaily等）
- ✅ 不受国内网络限制

### 场景2：中文查询技术趋势

**用户查询**："2024可持续商业空间设计趋势"

**处理流程**：
1. 检测到中文查询
2. ✅ 使用Tavily搜索全球网站
3. Tavily匹配"sustainable commercial space design 2024"
4. 返回全球最新趋势报告

**结果**：
- ✅ 获取国际权威设计媒体的2024趋势分析
- ✅ 覆盖欧美、亚洲的最新可持续设计实践
- ✅ 避免仅限于国内案例的局限性

### 场景3：仍需博查的情况

**用户查询**："国内零售设计行业政策文件2024"

**处理流程**：
1. LLM理解需求是"国内政策"
2. ✅ 主动选择博查工具（通过工具描述判断）
3. 博查搜索国内官方网站

**结果**：
- ✅ 获取国内相关政策文件
- ✅ 博查在此场景下仍有价值
