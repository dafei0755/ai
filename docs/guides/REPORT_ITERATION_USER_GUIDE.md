# 📖 报告迭代功能使用指南

## 快速开始

新增的三个功能会在报告生成后自动出现（如果有相关数据）：

### 1. 🔍 审核反馈与迭代改进

**位置**：报告底部的"审核反馈与迭代改进"折叠面板

**包含内容**：
- **迭代总结**：审核轮次、问题数量、解决率统计
- **红队质疑**：发现的所有风险点及响应措施
- **蓝队验证**：优化建议及价值增强点
- **评委裁决**：专业裁决和行动计划
- **甲方决策**：最终决策和实施计划

**示例场景**：
```
您想了解"为什么报告中采用了清水混凝土的1:1样板墙方案？"

→ 打开"红队质疑"标签
→ 找到"R1: 清水混凝土施工风险高"
→ 查看响应措施："强制要求1:1样板墙验证"
→ 查看状态："已修复"
```

---

### 2. 📝 用户访谈记录（校准问卷）

**位置**：报告中部的"用户访谈记录（校准问卷）"折叠面板

**包含内容**：
- 您在校准问卷中的所有回答
- 每个问题的上下文说明
- 从回答中提取的关键洞察

**示例场景**：
```
项目启动3个月后，您想回顾"当初对安藤忠雄风格的偏好是什么？"

→ 打开"用户访谈记录"
→ 查看Q1的回答："希望保留极简主义精神，但增加一些温暖元素"
→ 确认需求是否被正确实现
```

**价值**：
- ✅ 需求追溯：永久保存您的真实想法
- ✅ 团队对齐：设计师/工程师可随时查看原始需求
- ✅ 变更管理：对比当前方案与初始需求的差异

---

### 3. 📊 多轮审核可视化

**位置**：报告底部的"多轮审核可视化"折叠面板

**包含内容**：
- 总审核轮次、最终决策、改进率
- 各轮的红队/蓝队/评委评分
- 发现问题数和解决问题数
- 评分趋势柱状图

**示例场景**：
```
您想快速了解"项目质量是如何迭代提升的？"

→ 打开"多轮审核可视化"
→ 查看改进率：+23%
→ 查看评分趋势图：第1轮70分 → 第2轮82分
→ 查看问题统计：发现7个问题，解决5个
```

**解读技巧**：
- **改进率 > 20%**：说明审核效果显著
- **最终决策 = "通过"**：方案已达到发布标准
- **评分持续上升**：团队迭代能力强

## 常见问题

### Q1: 为什么我的报告没有显示审核反馈？
**A**: 只有经过多视角审核的报告才会包含审核反馈。如果是首次生成，可能还没有进入审核流程。

### Q2: 问卷回答可以修改吗？
**A**: 问卷回答提交后会永久保存，建议在提交前仔细检查。如需修改，请重新启动分析流程。

### Q3: 如何导出审核反馈数据？
**A**: 当前版本支持在"查看完整数据"中看到JSON格式的原始数据。未来版本将支持PDF/Excel导出。

### Q4: 评分趋势图看不清怎么办？
**A**: 可以点击右上角的"全屏"按钮放大查看，或在"查看完整数据"中获取原始数值。

### Q5: 如何理解"改进率"指标？
**A**: 改进率 = (最终评分 - 初始评分) / 初始评分。正值表示质量提升，负值表示质量下降。
