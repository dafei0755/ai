## ⚙️ 配置参数优化

### 信息充足阈值调整

```python
# 扩展前（v7.620）
INFO_SUFFICIENT_THRESHOLD = 0.40  # 过低，导致误判"信息充足"
INFO_ELEMENT_MIN_COUNT = 2        # 过低，关键信息可能缺失

# 扩展后（v7.800）
INFO_SUFFICIENT_THRESHOLD = 0.50  # +25%，减少误判
INFO_ELEMENT_MIN_COUNT = 3        # +50%，确保信息完整性
```

**原因**:
- 场景6（自闭症儿童）如果只检测到2个维度，评分0.35，会错误判断为"不足"
- 新阈值0.50 + 最少3个元素，能更准确识别关键信息
