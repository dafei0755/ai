# 任务梳理生成方法与依据 - 完整总结

## 快速回答

### 生成方法
**混合生成策略（Hybrid Generation）** = LLM两阶段生成 + 规则引擎补充 + 智能融合

### 生成依据
1. **复杂度分析**（6个维度评分）→ 推荐任务数量
2. **12维特征向量**（隐式识别设计模式）→ 调整任务结构
3. **10 Mode Engine**（设计模式层，理论基础）→ 指导类别选择
4. **Few-shot学习**（参考案例）→ 提升任务质量
5. **质量验证**（类别数量、任务分布）→ 确保合理性

---

## 一、完整生成流程（5步）

```
用户输入："四川广元苍溪云峰镇狮岭村，农耕文化，安藤忠雄的精神性空间..."
  ↓
【Step 1】复杂度分析
  - 输入长度: 62字符 → 0.4分
  - 信息维度: 6个 → 0.24分
  - 特殊场景: 2个 → 0.10分
  - 总复杂度: 0.36 → 推荐28-36个任务
  ↓
【Step 2】LLM两阶段生成
  Phase 1: 生成8-10个标准类别
    - 场地调研与分析（4个任务）
    - 文化背景与元素提炼（5个任务）
    - 设计对标与参考研究（4个任务）
    - ... 共8个类别
  Phase 2: 按类别生成详细任务
    - 狮岭村地形地貌与自然景观调研
    - 狮岭村农耕文化符号提取与空间转译
    - 安藤忠雄精神性空间案例研究
    - ... 共30个任务
  ↓
【Step 3】规则引擎生成（并行）
  - 基于project_features高分特征
  - 生成3-12个关键强制任务
  ↓
【Step 4】智能融合
  - 规则任务全部保留
  - LLM任务去重（相似度>0.70）
  - 优先级排序
  - 数量截断（如超过max_tasks）
  ↓
【Step 5】质量验证 + 动机标注
  - 检查类别数量（8-10个）
  - 检查任务分布（每类2-5个）
  - 标注12种动机类型
  ↓
输出：30个结构化任务
```

## 二、6个维度的复杂度评分

### 维度1：输入长度（权重15%）
```
< 50字符   → 0.2分
50-150字符  → 0.4分
150-300字符 → 0.6分
> 300字符   → 0.8分
```

### 维度2：信息维度数量（权重40%，最关键）
检测13个维度：
- 项目类型、预算约束、对标案例、文化要素
- 客群分析、核心张力、品牌主题、设计风格
- 功能需求、时间约束、地理位置、特殊场景
- 多业态融合

**重要**: 面积（㎡/平米）**不影响**复杂度

### 维度3：结构化数据深度（权重25%）
检查6个关键字段：
- design_challenge、character_narrative、physical_context
- analysis_layers、project_type、core_goals

### 维度4：特殊场景识别（权重15%）
- 诗意表达、多元对立、跨界融合、文化深度

### 维度5：句子数量（权重5-10%）
- ≥5个句子 → +0.10分

### 维度6：具体数字（权重10%）
- ≥3个数字 → +0.10分

### 任务数量映射
```
复杂度 0-0.12   → 8-13个任务
复杂度 0.12-0.25 → 18-23个任务
复杂度 0.25-0.45 → 28-36个任务
复杂度 0.45-1.0  → 40-52个任务
```

## 三、10 Mode Engine与任务生成的关系

### 核心关系
**10 Mode Engine** 是任务梳理的**理论基础**和**上游决策依据**

```
10种设计模式
  ↓（通过12维特征向量识别）
模式标签（如M5乡建在地型）
  ↓（调整任务结构）
类别选择 + 权重分配 + 任务粒度
  ↓
生成28-52个任务
```

### 10种模式与任务生成策略

| 模式 | 关键词 | 重点类别 | 任务数调整 |
|------|--------|---------|-----------|
| M1 概念驱动型 | 精神、表达、文化主题 | 文化提炼(5-6)、设计对标(4-5) | 弱化商业运营 |
| M2 功能效率型 | 办公、效率、动线 | 用户研究(4-5)、空间策略(5-6) | 弱化文化提炼 |
| M3 情绪体验型 | 酒店、体验、沉浸 | 用户研究(4-5)、空间策略(5-6) | 新增情绪节奏 |
| M4 资产资本型 | 商业、投资、回报 | 用户研究(4-5)、商业运营(5-6) | 新增资产分析 |
| M5 乡建在地型 | 乡村、在地、传统 | 场地调研(5-6)、文化提炼(5-6) | 新增产业闭环 |
| M6 城市更新型 | 老街区、改造、片区 | 场地调研(5-6)、商业运营(5-6) | 新增城市IP |
| M7 技术整合型 | 智能、AI、系统 | 技术材料(6-7)、创新策略(4-5) | 新增智能系统 |
| M8 极端环境型 | 高原、极寒、海岛 | 场地调研(5-6)、技术材料(6-7) | 新增抗性系统 |
| M9 社会结构型 | 家庭、代际、隐私 | 用户研究(5-6)、空间策略(5-6) | 新增社会关系 |
| M10 未来推演型 | 未来、趋势、演进 | 用户研究(4-5)、创新策略(5-6) | 新增趋势推演 |

### 12维特征与10模式的映射

| 特征维度 | 主要对应模式 | 次要对应模式 |
|---------|------------|------------|
| cultural | M1概念驱动、M5乡建在地 | M6城市更新 |
| commercial | M4资产资本 | M6城市更新 |
| emotional | M3情绪体验 | M1概念驱动 |
| technical | M7技术整合、M8极端环境 | M10未来推演 |
| sustainable | M5乡建在地、M8极端环境 | M10未来推演 |
| social | M9社会结构 | M6城市更新 |
| functional | M2功能效率 | M7技术整合 |
| professional | M2功能效率、M4资产资本 | M7技术整合 |

## 四、8个标准类别参考

### 类别1：场地调研与分析（4-5个任务）
- 地形、气候、景观、交通等
- 适用模式：M5乡建、M6城市更新、M8极端环境

### 类别2：文化背景与元素提炼（4-5个任务）
- 历史、传统、符号、叙事等
- 适用模式：M1概念驱动、M5乡建在地

### 类别3：用户需求与行为研究（3-4个任务）
- 目标客群、使用场景、体验需求等
- 适用模式：M2功能效率、M3情绪体验、M4资产资本

### 类别4：设计对标与参考研究（3-4个任务）
- 案例分析、设计师风格、创新趋势等
- 适用模式：M1概念驱动、M3情绪体验

### 类别5：概念设计与空间策略（4-5个任务）
- 空间组织、功能布局、动线设计等
- 适用模式：所有模式

### 类别6：技术系统与材料策略（3-4个任务）
- 结构、材料、设备、节能等
- 适用模式：M7技术整合、M8极端环境

### 类别7：商业模式与运营策略（2-3个任务）
- 盈利模式、运营管理、成本控制等
- 适用模式：M4资产资本、M5乡建在地、M6城市更新

### 类别8：可持续性与创新策略（2-3个任务）
- 生态、低碳、智能化等
- 适用模式：M5乡建在地、M7技术整合、M10未来推演

## 五、关键设计原则

### 1. 层次清晰
- **类别名称**：简洁通用（6-10字）
  - ✅ "场地调研与分析"
  - ❌ "狮岭村农耕文化与传统工艺挖掘"
- **任务标题**：具体化（包含项目特定信息）
  - ✅ "狮岭村地形地貌与自然景观调研"
  - ❌ "场地调研与分析"

### 2. 粒度适中
- 每个类别2-5个任务（灵活调整）
- 重要类别（文化、场地）4-5个任务
- 简单类别（商业、可持续）2-3个任务

### 3. 逻辑有序
- 按设计流程排序：调研分析 → 策略定位 → 概念设计 → 技术实施

### 4. 可执行性
- 任务描述包含具体行动
- 明确罗列调研维度
- 避免模糊表述

### 5. 质量保障
- 自动质量验证机制
- Few-shot学习增强
- 混合生成策略
- 智能去重和融合

## 六、示例：狮岭村民宿项目

### 输入
```
四川广元苍溪云峰镇狮岭村，农耕文化，安藤忠雄的精神性空间，
隈研吾的材料诗性，刘家琨的地域建筑智慧，新农村民宿集群
```

### 分析过程

**Step 1: 复杂度分析**
```
输入长度: 62字符 → 0.4分
信息维度: 6个（地理位置、文化要素、对标案例×3、项目类型）→ 0.24分
特殊场景: 2个（文化深度、跨界融合）→ 0.10分
总复杂度: 0.36 → 推荐28-36个任务
```

**Step 2: 模式识别**
```
主模式: M5_vernacular_rural (乡建在地型) - 得分0.82
次模式: M1_concept_driven (概念驱动型) - 得分0.68
```

**Step 3: 类别调整**
```
标准8类别 → 调整为9类别

1. 场地调研与分析: 4个 → 5个（强化村落肌理）
2. 文化背景与元素提炼: 4个 → 6个（强化地域文化+建筑师风格）
3. 用户需求与行为研究: 3个 → 3个
4. 设计对标与参考研究: 4个 → 5个（强化建筑师案例）
5. 概念设计与空间策略: 4个 → 4个
6. 技术系统与材料策略: 4个 → 5个（强化本地材料）
7. 商业模式与运营策略: 3个 → 4个（强化产业嵌入）
8. 可持续性与创新策略: 3个 → 2个
9. 乡村产业与经济闭环: 0个 → 3个（新增）

总计: 37个任务
```

**Step 4: 生成任务示例**

```
类别1: 场地调研与分析（5个任务）
1. 狮岭村地形地貌与自然景观调研
2. 云峰镇气候特征与微气候分析
3. 场地交通可达性与周边资源评估
4. 村落空间肌理与建筑现状调研
5. 狮岭村水系分布与生态环境评估

类别2: 文化背景与元素提炼（6个任务）
6. 狮岭村农耕文化符号提取与空间转译
7. 传统工艺与手工艺资源调研
8. 地方民俗与生活方式研究
9. 文化叙事线索梳理与故事化设计
10. 非物质文化遗产保护与活化策略
11. 农耕文化在空间中的当代表达方式

类别4: 设计对标与参考研究（5个任务）
12. 安藤忠雄精神性空间案例研究：建筑哲学、光影运用、清水混凝土美学
13. 隈研吾材料诗性与在地性表达分析：负建筑理念、材料创新、自然融合
14. 刘家琨乡村建筑实践解读：地域建筑探索、低技策略、乡土材料运用
15. 国内外乡村民宿集群优秀案例对标
16. 建筑师风格在乡村语境中的转译策略

... 共37个任务
```

**Step 5: 质量验证**
```
✅ 任务质量验证通过: 9个类别，分布均衡
✅ 类别数量: 9个（在8-10个范围内）
✅ 任务分布: 每个类别2-6个（在2-5个范围内，略超）
```
