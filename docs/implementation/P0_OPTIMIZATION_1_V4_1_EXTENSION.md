## 🎨 示例设计策略

### 示例1: v4_1_targeted_competitor_001（4240字符）
**场景**: 针对性问答 - 竞品空间设计特色
**用户请求**: "星巴克的空间设计有哪些值得学习的特色？"

**设计亮点**:
1. **深度拆解4个核心设计模式**:
   - 分区灵活性（Solo/Social/Work/Quick区）
   - 感官品牌化（视觉/嗅觉/听觉/触觉）
   - 动线与视觉引导（体验最大化原则）
   - 本地化与社区连接（反同质化策略）

2. **数据驱动分析**:
   - 停留时长75分钟（vs快餐店25分钟）
   - 座位周转率≥3次/天
   - NPS净推荐值77（行业平均55）
   - 本地化门店忠诚度高35%

3. **可迁移洞察提炼**:
   - 4个actionable insights（用户画像锚定/感官一致性/慢体验/本地化免疫系统）
   - 每个洞察包含原理+建议，直接可用于其他项目

### 示例2: v4_1_comprehensive_tsutaya_001（3185字符）
**场景**: 完整报告 - 茑屋书店案例深度拆解
**用户请求**: "请提供茑屋书店（代官山T-SITE）的完整设计研究报告，提取可复用的设计模式"

**设计亮点**:
1. **完整研究方法论展示**:
   - 文献研究（23篇文献）
   - 实地观察（3次×4小时，127位顾客）
   - 专家访谈（2位专家+1位从业者）
   - 对比分析（10家标杆书店）

2. **5大关键发现**:
   - 空间作为"时间容器"（126分钟停留时长）
   - 策展式空间叙事（场景组织vs分类法）
   - 混合业态化学反应（功能溶解设计）
   - 慢设计物理实现（书架1.8m，通道2.5m，照度300-400lux）
   - T字型建筑动线策划（探索新鲜感）

3. **三层级设计启示**:
   - 战略层：从销售商品→销售时间
   - 战术层：策展思维重构组织逻辑
   - 执行层：反效率设计物理参数清单

4. **证据基础完整性**:
   - 一手数据（观察记录+顾客访谈+空间测量）
   - 二手资料（品牌白皮书+建筑评论+学术研究）
   - 专家验证（交叉确认设计策略）

### 示例3: v4_1_targeted_research_method_001（7232字符）⭐最长示例
**场景**: 针对性问答 - 用户行为研究方法论
**用户请求**: "如何研究用户在共享办公空间的真实使用行为？需要采用哪些研究方法？"

**设计亮点**:
1. **5种研究方法系统化讲解**:
   - 行为观察法（时间采样+关键事件，成本低）
   - 深度访谈法（20-30人，理论饱和，成本中）
   - 问卷调查法（≥500份，统计显著，成本低）
   - 日志记录法（10-15人×2周，捕捉碎片体验，成本中）
   - 空间实验法（对照实验，验证改进方案，成本中高）

2. **每个方法包含完整执行指南**:
   - 研究目标（Why）
   - 执行步骤（How）- 详细到操作层
   - 预期输出（What）- 具体交付物
   - 资源需求（人力/工具/成本）

3. **实用策略指导**:
   - 预算受限策略（<3万元，组合观察+问卷）
   - 时间受限策略（<2周，快速问卷+专家走查）
   - 深度研究策略（>5万元/>6周，全方法组合）

4. **风险缓解方案**:
   - 隐私顾虑→知情同意+数据匿名
   - 霍桑效应→隐蔽观察+长周期
   - 回收率低→激励机制+多渠道

## 💡 关键洞察

### 1. 示例长度差异很大（3185-7232字符）
**原因**: 研究方法论示例需要详细的执行指南，自然更长。
**启示**: 不同类型的示例有不同的最佳长度，不应强制统一。方法论类示例可以更长（6000-8000字符），具体案例类示例适中（3000-5000字符）。

### 2. V4_1匹配targeted示例更频繁
**数据**: 3个测试案例中，2个匹配到targeted示例。
**原因**: V4_1的用户通常提出针对性研究问题（"如何分析XX"/"YY有什么特点"）。
**建议**: V4_1示例库应保持2:1的targeted:comprehensive比例。

### 3. 研究方法论示例通用性强
**发现**: v4_1_targeted_research_method_001可适用于多种场景（共享办公→酒店→餐厅）。
**价值**: 一个好的方法论示例=N个具体案例示例的通用能力。
**策略**: 优先创建1-2个通用方法论示例，再补充3-5个具体案例。

### 4. 证据基础模板很重要
**观察**: 茑屋示例的"evidence_base"字段成为V4_1的独特价值展示。
**效果**: 让LLM学会"研究报告需要引用证据"这一学术规范。
**推广**: 其他角色的comprehensive示例也应包含"证据基础"部分。
