# 四川广元苍溪云峰镇狮岭村

# 新农村建设升级示范模型（L5 升级版）

------

# 一、总命题升级

## 核心命题：

> 乡村不是被拯救的对象，而是被重新组织的生产结构。

------

## 狮岭模式 1.0

### 果园型乡村文明再生产模型

我们不做“民宿项目”。

我们做的是：

- 农业结构升级
- 空间结构再组织
- 文化叙事重建
- 金融逻辑嵌入
- 治理结构重塑

------

# 二、双核概念系统（文化核 + 产业核）

## 文化核：

**沉静的丰饶**

- 拒绝乡村主题公园化
- 拒绝城市奢侈品复制
- 构建安静而自信的乡村空间

------

## 产业核：

**果园型经济模型**

苍溪的真正核心资源不是“乡村”。

而是：

- 红心猕猴桃原产地
- 梯田型丘陵地貌
- 水汽充足的嘉陵江流域气候

因此我们将：

> 果园作为空间骨架，而不是背景。

------

## 三位一体结构：

| 层级 | 角色                  |
| ---- | --------------------- |
| 农业 | 果园成为主景观        |
| 空间 | 民宿嵌入果园          |
| 金融 | 果园认养 + 农产品订阅 |

------

# 三、空间总体规划结构

## 总体结构：

**散点聚落 + 果园骨架 + 精神中枢**

```
村口广场
   │
文化带
   │
精神核心建筑（光之院）
   │
果园型民宿聚落
   │
农业体验系统
   │
后勤与村民生活区
```

> 4.5-6年回收周期

# 十、50年视角

狮岭模式不是短期网红。

它必须满足：

- 建筑50年可持续
- 材料自然老化
- 产业自循环
- 文化可传承

------

# 十一、风险声明（必须锋利）

如果：

- 项目沦为网红打卡地
- 设计符号压倒村庄真实生活
- 农业被边缘化成为景观背景

那么：

> 狮岭模式宣告失败。

------

# 十二、最高形态判断

当：

- 村民主动讲述自己的故事
- 果园成为生产与风景的统一体
- 建筑像梯田一样自然生长
- 游客不再拍照，而是停留

那时：

> 狮岭村不再是“项目”。
> 它成为一种新的乡村范式。

------

# 终极总结

狮岭模式 1.0

不是民宿集群。

它是：

- 农业结构升级模型
- 空间文明再组织模型
- 村庄治理再分配模型
- 果园型经济实验模型

如果成功，

它将成为：

> 中国丘陵型乡村再生产的标准样本。
