

# 狮岭“光之院”

## 精神核心建筑 · 施工级深化方案 v1.0

------

# 一、建筑定位（真实角色确认）

### 功能身份（不是象征物）

1. 村庄文化馆
2. 农耕展示空间
3. 建筑论坛举办空间
4. 冬季室内活动空间
5. 小型仪式场（婚礼 / 收获节）

------

### 建筑面积建议

- 建筑总面积：480–650㎡
- 单层为主，局部下沉
- 局部夹层（设备/储藏）

------

# 二、总体空间构成（可施工平面逻辑）

## 平面结构（环院型）

```
入口灰空间
      ↓
前厅（压缩）
      ↓
下沉光庭（核心）
      ↓
环绕展厅
      ↓
茶室 / 讲座区
      ↓
后勤区
```

------

## 空间尺寸建议

| 空间     | 建议尺寸  |
| -------- | --------- |
| 下沉庭院 | 10m × 10m |
| 展厅环廊 | 宽 3m     |
| 主展厅   | 8m × 12m  |
| 多功能厅 | 9m × 12m  |
| 茶室     | 6m × 8m   |
| 屋檐深度 | ≥1.2m     |

------

# 三、核心空间：下沉光庭

这是施工难点。

------

## 结构系统

- 结构形式：钢筋混凝土框架 + 局部剪力墙
- 下沉深度：1.2m–1.5m
- 外墙：300mm清水混凝土
- 防水等级：一级防水

------

## 光缝设计

- 屋面开设 300mm–600mm 光缝
- 采用夹胶中空钢化玻璃
- 内侧设排水槽

## 排水系统

必须极其严谨：

- 下沉庭院四周设隐形排水沟
- 地面微坡 1.5%
- 雨水接入地下蓄水池
- 设置溢流口

------

# 四、结构系统深化

## 基础

- 独立基础 + 地梁
- 山地需做地勘
- 基础埋深 ≥ 1.8m

------

## 墙体

| 类型       | 厚度             |
| ---------- | ---------------- |
| 外墙       | 300mm 清水混凝土 |
| 内墙       | 200mm 混凝土     |
| 夯土包裹层 | 150mm（非承重）  |

------

## 屋面系统

- 现浇钢筋混凝土屋面
- 屋面保温层
- 防水卷材双层
- 屋面覆土 100mm（局部绿化）

------

# 五、材料系统（可落地做法）

## 清水混凝土做法

- 竹胶板模板
- 对拉螺杆孔统一布点
- 预留灯具暗盒
- 表面防水透明保护剂

------

## 夯土墙（内侧展示）

- 稳定土配比：土 + 石灰 + 5%水泥
- 每层夯筑厚度 150mm
- 表面保留手工纹理

------

## 地面

| 区域     | 材料     |
| -------- | -------- |
| 下沉庭院 | 毛面石板 |
| 展厅     | 水磨石   |
| 茶室     | 实木地板 |
| 外廊     | 防滑青石 |

------

# 六、机电系统整合

## 空调系统

- 风管机
- 隐藏于环廊吊顶
- 送风口线性隐藏

------

## 地暖系统

- 冬季湿冷必须考虑
- 水地暖 + 空气能

------

## 照明系统

- 无主灯设计
- 洗墙灯隐藏于地面
- 光庭自然光为主

------

# 七、消防规范控制

- 单层为主，建筑高度 < 12m
- 两个疏散出口
- 环形疏散动线
- 耐火等级二级以上

------

# 八、成本控制测算

## 造价估算（粗算）

| 项目       | 单方造价       |
| ---------- | -------------- |
| 主体结构   | 1600–2000元/㎡ |
| 清水混凝土 | +400元/㎡      |
| 机电系统   | 600元/㎡       |
| 室内精装   | 1000–1500元/㎡ |

------

### 总造价估算：

约 3500–4500元/㎡

若建筑 600㎡：

约 210万 – 270万

可控。

------

# 九、施工风险提示

1. 清水混凝土必须找有经验团队
2. 山地排水必须优先设计
3. 光缝必须防渗漏
4. 下沉庭院必须防积水

------

# 十、施工阶段建议

### 阶段1

- 结构封顶
- 光庭成型
- 外立面完成

### 阶段2

- 室内精装
- 灯光调试
- 机电测试

------

# 十一、最终空间效果判断标准

不是看照片。

而是判断：

1. 正午是否能形成明确光柱？
2. 阴天是否仍有精神感？
3. 雨天是否排水顺畅？
4. 冬天是否舒适？
5. 施工是否可控？

------

精神性不是造型。
