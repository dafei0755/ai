# 四川广元苍溪云峰镇狮岭村 — 新农村建设升级示范案例

## 12 Ability Core 全维度解析

---

> **项目定位**：以狮岭村为载体，打造具有文化示范意义的民宿集群，深度挖掘在地农耕文化、产业结构与乡村经济逻辑，融合安藤忠雄的精神性空间、隈研吾的材料诗性，以及刘家琨、王澍、谢柯等中国建筑师的地域智慧，形成商业成功与文化深度兼具的项目模型。

---

## 一、项目背景与地域基因

### 1.1 地理坐标

- **位置**：四川省广元市苍溪县云峰镇狮岭村
- **地貌**：川北丘陵地带，嘉陵江流域，山地与河谷交错
- **气候**：亚热带湿润季风气候，夏热冬寒，年降雨量约1000mm
- **海拔**：约400-800m，山势缓坡

### 1.2 文化基因

- **农耕传统**：苍溪为红心猕猴桃原产地，雪梨种植历史悠久，传统稻作文化
- **历史脉络**：川北古蜀道交通节点，红色革命文化（红四方面军长征经过）
- **民居特征**：川北穿斗式木结构，青瓦坡屋顶，院落围合，夯土墙基
- **手工艺**：竹编、草编、土陶、酿造（醋/酒）
- **村落肌理**：依山而建，顺地形展开，散点式聚落，沿溪分布

### 1.3 产业结构

- **农业主业**：猕猴桃、雪梨、水稻、油菜
- **副业**：养殖（跑山鸡、黑猪）、竹林经济
- **潜在增长**：农文旅融合、民宿经济、电商农产品

## 二、12 Ability Core 逐模块解析

---

### 能力一｜概念建构能力（Concept Architecture）

#### 核心命题

**"归田不归旧"**

——在乡村的物理环境中，构建一种既不怀旧、也不都市化的第三种栖居状态。

#### 五层结构解析

**第一层｜抽象源提取**

- 精神核心：**"城市游子的回归焦虑与乡村自信的重建"**
- 真正张力：现代人渴望乡土根性，但无法接受落后生活条件；乡村渴望振兴，但不愿沦为城市消费品
- 存在焦虑：归属感缺失 × 身份认同撕裂

**第二层｜核心命题凝缩**

> **"沉静的丰饶"**

- 方向性：空间必须安静、内敛、不张扬
- 限制性：禁止奢华堆砌，禁止乡村主题公园化

**第三层｜结构规则生成**

| 规则维度 | 具体约束 |
|---------|---------|
| 尺度规则 | 建筑体量不超过两层，屋脊线不高于现有最高民居 |
| 材料规则 | 主材限定3种：夯土/竹/青瓦，辅材限定2种：清水混凝土/原木 |
| 光线规则 | 自然光为主，人工光仅补充；公共空间侧光为主，禁止顶部大面积采光 |
| 动线规则 | 步行优先，车行止于村口，内部以步道+田埂连接 |
| 边界规则 | 建筑与自然无硬边界，檐下空间作为过渡灰空间 |
| 留白规则 | 每栋建筑保留不少于30%的"空"——庭院、檐廊、天井 |

**第四层｜一致性校验**

概念"沉静的丰饶"必须贯穿：
- 平面：院落围合感
- 立面：低调、水平延展
- 软装：土布、陶器、编织物
- 灯光：暖色温、低照度
- 景观：保留原有农田肌理
- 细部：铜件替代不锈钢，手工质感替代机器精度

**第五层｜叙事表达**

对甲方（村委/政府）：这是一个"让村庄自己说话"的项目，不是外来者的审美植入
对运营方：概念"沉静的丰饶"天然适合中高端客群定价，避免价格战
对施工方：减少材料种类 = 降低施工复杂度 = 可控成本

#### 概念成熟度：L4 系统概念

概念不仅是美学语言，而是整合了空间、经济、文化的系统命题。

### 能力二｜空间结构能力（Spatial Structuring）

#### 五层结构解析

**第一层｜平面逻辑系统**

采用"散点聚落 + 线性串联"结构：

```
村口广场（集散）
  │
  ├── 公共文化带（线性）
  │     ├── 农耕文化馆
  │     ├── 手工艺工坊
  │     └── 公共茶室 / 咖啡
  │
  ├── 民宿聚落A（山腰·3-5栋）
  │     └── 独立院落，朝向河谷
  │
  ├── 民宿聚落B（坡地·3-5栋）
  │     └── 台地错层，面向梯田
  │
  ├── 农业体验区（保留农田）
  │     ├── 猕猴桃采摘园
  │     └── 稻田体验
  │
  └── 后勤服务区（隐藏）
        ├── 员工宿舍
        ├── 洗衣/仓储
        └── 设备间
```

**主动线**：村口 → 文化带 → 民宿聚落（步行约8-12分钟）
**后勤动线**：独立道路，不与客人动线交叉
**车行动线**：止于村口停车场，电瓶车入村

**第二层｜尺度控制系统**

- **压缩**：入口村道窄、低，营造期待感
- **释放**：到达聚落后，庭院打开，视线延展至河谷/梯田
- **呼吸节奏**：窄巷 → 院落 → 窄巷 → 公共广场 → 田野

**第三层｜分区与层级系统**

| 层级 | 区域 | 属性 |
|------|------|------|
| 公共区 | 村口广场、文化馆、公共茶室 | 完全开放 |
| 半公共区 | 民宿公共庭院、餐厅 | 住客共享 |
| 半私密区 | 民宿走廊、檐廊 | 视线可及但不打扰 |
| 私密区 | 客房内部、私人庭院 | 完全独立 |

**第四层｜体量与秩序系统**

- 主从关系：公共文化建筑体量最大（但仍然低调），民宿体量小而散
- 秩序类型：**散点型**——顺应地形，不强加轴线
- 参照：王澍文村改造的"尊重聚落自然生长逻辑"

**第五层｜结构落地**

- 梁柱系统：穿斗式木构为主，局部钢结构加固
- 基础：夯土/毛石基础，适应山地地形
- 消防：每栋独立，间距满足防火要求
- 管线：地下管网统一规划，避免外露

#### 空间结构成熟度：L4 复杂系统整合

### 能力三｜叙事节奏能力（Narrative Orchestration）

#### 体验时间线设计

**整体叙事弧线**：

```
到达 → 压缩 → 发现 → 豁然 → 沉浸 → 回味

城市   村口    小巷   庭院    田野    离开
噪杂   收窄    幽暗   打开    辽阔    携记忆归
```

**第一层｜入口仪式**

- 从公路转入村道，路面变窄，两侧竹林遮蔽
- 光线从明到暗，声音从车噪到鸟鸣
- 到达村口广场前，经过一段"竹隧道"——心理从城市切换到乡村

**第二层｜过渡铺垫**

- 村口广场不大，石板地面，一棵老树
- 视线被建筑半遮挡，不能一眼看到全貌
- 沿文化带缓行，每走一段有一个"发现"——老农具墙、手工艺展示、水渠声

**第三层｜高潮节点**

- 到达民宿聚落时，庭院打开
- 视野突然释放——面前是层层梯田或河谷远山
- 这是"记住的瞬间"
- 建筑设计在此处使用大面玻璃或檐下框景

**第四层｜沉静回落**

- 客房内部：低照度、木质温度、土墙质感
- 庭院中的水声、虫鸣
- 不再有刺激，只有安静

**第五层｜记忆锚点**

- 清晨推门见雾中梯田
- 傍晚檐下看炊烟
- 夜晚天井看星空
- 这些都是"带走的画面"

#### 叙事节奏成熟度：L4 情绪结构化

### 能力四｜材料系统能力（Material Intelligence）

#### 材料哲学

> **"就地生长"** —— 材料应像植物一样属于这片土地

#### 材料等级系统

| 等级 | 材料 | 用途 | 来源 |
|------|------|------|------|
| 主材 | 夯土 | 墙体主体 | 本地取土，现代夯筑工艺 |
| 主材 | 竹 | 屋架/隔断/装饰 | 本地竹林 |
| 主材 | 青瓦 | 屋面 | 本地窑烧 |
| 辅材 | 清水混凝土 | 基础/结构节点 | 安藤忠雄精神性转译 |
| 辅材 | 原木（杉木） | 门窗/家具 | 本地林场 |
| 点缀材 | 铜/黄铜 | 五金/灯具 | 手工锻造 |
| 点缀材 | 土陶 | 器皿/装饰 | 本地工艺 |

#### 材料过渡系统

- 室外到室内：夯土墙从外延续到内，但内墙做细磨处理
- 地面过渡：院落石板 → 室内水磨石 → 卧室原木地板
- 墙到顶：夯土墙 → 竹编吊顶 → 木椽裸露

#### 材料与建筑师精神的转译

| 建筑师 | 精神 | 材料转译 |
|--------|------|---------|
| 安藤忠雄 | 光的精神性 | 清水混凝土窄缝引光，创造"光的通道" |
| 隈研吾 | 材料的消解 | 竹编表皮让建筑轻盈，与环境融合 |
| 王澍 | 传统工艺复兴 | 夯土墙、瓦片砌筑、手工粗糙质感 |
| 刘家琨 | 低技策略 | 本地工匠可操作的简单构造 |
| 谢柯 | 在地温度 | 家具使用本地旧木改造，保留时间痕迹 |

#### 性能与寿命

| 材料 | 风险 | 应对措施 |
|------|------|---------|
| 夯土 | 防水性差 | 添加石灰/水泥稳定剂，外加防水檐 |
| 竹 | 虫蛀/霉变 | 碳化处理，烟熏防虫 |
| 青瓦 | 渗漏 | 双层瓦+防水卷材底层 |
| 原木 | 受潮变形 | 烘干处理，架空底部 |

#### 材料系统成熟度：L4 材料逻辑统一

### 能力五｜灯光系统能力（Lighting Architecture）

#### 灯光哲学

> **"日出而作，日落而息"** —— 灯光跟随自然节律，而非对抗黑暗

#### 五层灯光系统

**第一层｜基础照明**

- 公共区：环境光以壁灯为主，避免筒灯矩阵
- 客房：无主灯设计，分区照明
- 后勤区：功能性照明，简洁高效

**第二层｜层级系统**

| 层级 | 照度 | 位置 |
|------|------|------|
| 基础层 | 30-50lux | 走廊、过渡空间 |
| 中间层 | 80-150lux | 餐厅、公共活动区 |
| 焦点层 | 200-300lux | 展品照明、阅读区 |

**第三层｜情绪光控制**

- 色温统一：2700K-3000K暖色温为主
- 公共茶室：烛光色温（2200K），营造围炉感
- 文化馆：混合色温，展品区3500K中性光

**第四层｜昼夜节律**

- 白天：最大化自然光利用，窗/天窗/光缝
- 黄昏：自动过渡到暖光，模拟"炊烟时分"
- 夜晚：极低照度，仅路径引导光+庭院微光
- 清晨：东向窗允许晨光唤醒

**第五层｜行为引导**

- 村口到民宿的步道：地面嵌入式微光引导
- 公共空间：光池效应引导停留
- 田野路径：不设照明，鼓励星空体验

#### 灯光成熟度：L3 情绪表达

### 能力九｜社会关系建模能力（Social Structure Modeling）

#### 三重社会关系

**关系一｜村民与项目**

- 核心原则：**村民是受益者，不是被观赏者**
- 就业结构：管家/厨师/手艺展示 = 本地村民优先
- 收益共享：村集体占股 + 农产品供应链
- 文化参与：老手艺人入驻工坊，技艺传承

**关系二｜游客与乡村**

- 避免"人类动物园"式观光
- 体验设计：参与式而非旁观式（下田/手作/共餐）
- 隐私分级：游客与村民生活区物理分隔
- 时间分区：体验活动时段与村民生活时段错开

**关系三｜游客与游客**

- 公共空间鼓励适度社交（公共茶室、田间餐桌）
- 客房保证绝对私密
- 活动可选参与，不强制社交

#### 冲突缓冲机制

| 潜在冲突 | 缓冲设计 |
|---------|---------|
| 游客噪音→村民生活 | 民宿聚落与村民区物理间距≥100m |
| 商业化→文化失真 | 文化展示由村民主导，非外来策划 |
| 淡季→村民收入不稳 | 保底工资 + 旺季分红机制 |
| 代际差异→年轻人外流 | 提供返乡就业岗位 + 技能培训 |

#### 关系演化适配

- 建筑可拆分/合并（客房可变为村民住房或工坊）
- 公共空间可转换功能（淡季为村民活动中心）
- 预留3-5栋扩展用地

#### 社会建模成熟度：L4 演化适配

### 能力十｜环境适应能力（Environmental Adaptation Intelligence）

#### 气候应对策略

**川北气候特征**

| 变量 | 数据 | 设计应对 |
|------|------|---------|
| 夏季高温 | 35℃+ | 大屋檐遮阳，夯土墙蓄冷，自然通风 |
| 冬季湿冷 | 0-5℃ | 地暖系统，保温层，南向开窗 |
| 年降雨 | ~1000mm | 坡屋顶排水，檐口深出800mm+ |
| 日照 | 偏少（盆地） | 最大化南向采光，避免遮挡 |
| 湿度 | 年均75%+ | 架空底层防潮，材料防霉处理 |

#### 被动式设计策略

- **自然通风**：南北对流窗 + 天井拔风效应
- **自然采光**：侧窗 + 高侧窗 + 光缝
- **隔热蓄冷**：600mm夯土墙热惰性
- **遮阳**：深檐 + 竹帘可调节
- **排水**：坡屋面 + 院落明沟 + 雨水花园

#### 能源系统

- 太阳能热水（屋顶集成，隐藏于坡屋面内）
- 空气能热泵（冬季供暖主力）
- 雨水收集（灌溉 + 景观用水）
- 生态污水处理（人工湿地）

#### 材料耐久匹配

| 环境因素 | 材料风险 | 应对措施 |
|---------|---------|---------|
| 高湿度 | 木材霉变 | 架空+通风+防霉处理 |
| 暴雨 | 夯土冲刷 | 深檐保护+防水基座 |
| 冻融 | 瓦片开裂 | 选用高温烧制工艺瓦 |
| 虫害 | 竹材虫蛀 | 碳化处理+定期维护 |

#### 环境适应成熟度：L3 被动式设计

### 能力十一｜运营与产品化能力（Operation & Productization Intelligence）

#### 产品体系

**标准化模块**

| 模块 | 标准面积 | 可复制性 |
|------|---------|---------|
| A型客房（标准院落） | 45㎡+15㎡庭院 | 高 |
| B型客房（亲子套房） | 65㎡+20㎡庭院 | 中 |
| C型客房（蜜月/VIP） | 80㎡+30㎡庭院 | 低（定制化） |
| 公共茶室单元 | 60㎡ | 高 |
| 手工艺工坊单元 | 40㎡ | 高 |

**运营动线**

- 清洁路径：集中时段（10:00-14:00），独立通道
- 餐饮供给：后厨→餐厅→田间餐桌（三条线路）
- 农产品：田间→加工间→展示/快递间

#### 品牌系统

- **视觉语言**：夯土色+竹纹理+手写体
- **材料基因**：夯土+竹+青瓦 = 三要素识别
- **灯光风格**：暖色温+低照度+烛光感
- **空间比例**：水平延展+低矮+围合

#### 可迭代更新

- 软装每3年更新一次（陶器/织物/植物）
- 竹编构件每5年维护/更换
- 智能系统每2年升级
- 景观随季节自然更新

#### 规模化路径

```
狮岭村（原型） → 苍溪县其他村（复制） → 广元市级示范 → 川北品牌
```

- 骨架标准（结构/材料/运营流程）
- 表皮可变（每村在地文化不同）

#### 产品化成熟度：L3 产品线模型

### 能力十二｜文明表达与跨学科整合能力（Civilizational Expression & Interdisciplinary Integration）

#### 文明议题

**核心议题：中国乡村的第三条路**

- 第一条路：完全城市化（消灭乡村）
- 第二条路：博物馆化（冻结乡村）
- **第三条路：活化——让乡村成为一种当代生活方式**

#### 抽象理论转译

| 理论 | 空间转译 |
|------|---------|
| 费孝通"乡土中国" | 差序格局 → 院落围合的隐私分级 |
| 安藤忠雄"间"的哲学 | 人与自然之间 → 檐下灰空间 |
| 隈研吾"负建筑" | 建筑消解于环境 → 材料与地形融合 |
| 王澍"自然建造" | 手工性 → 夯土墙面保留工匠手纹 |
| 谢柯"日常诗性" | 生活本身即美 → 保留真实农事场景 |

#### 跨学科整合

| 学科 | 整合维度 |
|------|---------|
| 社会学 | 村落权力结构与公共空间设计 |
| 经济学 | 乡村产业闭环与空间功能对应 |
| 人类学 | 农耕仪式与空间叙事节奏 |
| 生态学 | 人工湿地+雨水花园+林田系统 |
| 心理学 | 城市人"回归焦虑"的空间疗愈 |

#### 文化政治平衡

| 张力 | 平衡策略 |
|------|---------|
| 传统 vs 现代 | 传统材料+现代工艺 |
| 商业 vs 公共 | 商业收益反哺村集体 |
| 外来 vs 本土 | 大师精神内化，不做符号搬运 |
| 效率 vs 诗意 | 运营标准化，体验感性化 |

#### 50年视角

- 建筑材料选择经受时间考验（夯土/石/木≥50年）
- 功能可转换（民宿→居住→工坊）
- 村庄产业结构自我造血
- 文化基因随代际传承而非固化

#### 文明表达成熟度：L4 跨学科整合

## 五、实施路线图

### 第一阶段（第1-6个月）：规划与设计

- 详细地形测绘与村落肌理记录
- 在地文化深度调研（口述史、手艺人访谈、产业田调）
- 概念方案设计与村民/政府讨论
- 材料打样与工艺测试

### 第二阶段（第7-18个月）：一期建设

- 3-5栋示范民宿 + 公共茶室 + 手工艺工坊
- 基础设施（道路/管网/停车场）
- 景观初步营造

### 第三阶段（第19-30个月）：运营与完善

- 试运营 + 数据反馈 + 空间调整
- 品牌建立 + 媒体传播
- 村民培训 + 服务体系成熟

### 第四阶段（第31-48个月）：二期扩展

- 增加5-8栋民宿
- 农业体验区完善
- 研学/团建产品线开发

### 第五阶段（第49个月+）：模式输出

- 总结"狮岭模式"
- 向周边村落复制核心经验
- 建立区域品牌联盟
