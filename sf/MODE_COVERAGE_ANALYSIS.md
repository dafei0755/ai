## 二、真实空白：现有模式无法精准覆盖的问题群

### 2.1 【结构性空白 A】健康疗愈型设计 — 约 18 道题，完全没有专属模式

**识别信号**：问题核心是"心理安全感、疗愈节奏、恢复机制、降低恐惧"，而非体验感动或功能效率。

| 题号 | 问题核心 | 现有模式的误判风险 |
|------|----------|------------------|
| Q8 | 自闭症儿童感统空间 | M9会过度聚焦"权力关系"，忽略治疗性设计逻辑 |
| Q32 | 严重过敏儿童卧室（医疗级材料） | M7会偏向"系统技术"，忽略"非病房感" |
| Q44 | 正念冥想/睡眠焦虑空间 | M3会偏向"情绪体验"，忽略生理调节节律 |
| Q47 | 临终病房（尊严+疗愈） | M9/M3 均无法处理"死亡叙事"的空间逻辑 |
| Q53 | 牙科诊所减恐惧 | M2会偏向"效率流程"，忽略神经性恐惧控制 |
| Q58 | 高端月子中心 | M3未包含"脆弱生命保护"维度 |
| Q65 | 无障碍住宅（轮椅使用者） | M2偏效率，M9偏权力，但无障碍有独特的身体尊严逻辑 |
| Q80 | 养老社区打破孤岛感 | M9+M4 无法处理"代际焦虑"与疗愈叙事 |
| Q92 | 抗焦虑社交空间（高压金融人群） | M3"情绪体验"方向相反（追求高潮），本题核心是"降压" |
| Q101 | 青少年心理干预中心 | 无模式覆盖"羞耻感消除"的空间逻辑 |
| Q110 | 宠物医疗社交中心 | 无模式覆盖"宠物主人焦虑管理"维度 |
| Q115 | 实验室高压人员休息区 | M8偏向结构生存，无"认知恢复区" |
| Q119 | 银发再就业中心（尊严感） | M9无法处理老年人的社会尊严重建逻辑 |
| Q128 | 全球员工心理恢复中心 | M3偏向"体验高峰"，本题需要"下行放松" |
| Q133 | 战后重建社区恢复中心 | M8偏生存，但创伤敏感空间语言是独立的 |
| Q137 | 火星返回重力适应住宅 | 极端案例，但疗愈逻辑清晰 |
| Q175 | 高端康养小镇规划 | M4+M9混合，但医疗疗愈维度被忽略 |
| Q182 | 神经科学感知实验空间 | M10+M7 均不能处理"空间作为研究对象" |

**这类问题的独特设计逻辑：**
- **安全感优先于美学**——材料先问"无害"，再问"好看"
- **下行节奏**——不是叙事节奏的"高潮"，而是降低刺激、减少焦虑
- **无障碍身体尊严**——空间不暴露使用者的弱势状态
- **生理-心理双调节**——光节律、声音、材料触感需同时控制
- **从业者双视角**——设计必须同时服务于"服务提供者"和"被服务者"

### 2.2 【结构性空白 B】预算约束策略型设计 — 约 14 道题，现有模式有方向误判风险

**识别信号**：问题核心是"在严格成本上限内，如何配置预算以最大化空间价值感"，而非盈利逻辑。

| 题号 | 问题核心 | M4 的误判方向 |
|------|----------|--------------|
| Q33 | 8000元25㎡改造（刚毕业大学生） | M4关注溢价、客群资产；此题问的是极限压缩+价值杠杆 |
| Q42 | 贵州低预算青旅（背包客） | M5覆盖在地，但"低成本运营可持续"方法论缺失 |
| Q69 | 保障性住房标准（低收入家庭） | M2偏功能，M4偏商业；此题核心是"尊严+效率+成本" |
| Q77 | 50万上限上海老弄堂（全包） | M3偏体验，M4偏溢价；此题是**预算分配杠杆学** |
| Q78 | 3000元/㎡豪宅（低于刚需标准） | M4误判为"溢价"题，实际是"成本约束下的体面感" |
| Q83 | 可持续材料替换成本高 20% 的论证 | M10偏未来趋势；此题是**全生命周期成本模型** |
| Q84 | 30小时换装施工（不闭店） | 无模式覆盖"施工策略"与"时间极限约束" |
| Q93 | 废弃粮仓低成本孵化中心 | M6偏更新策略，无低成本工业美学方法论 |
| Q103 | 西部山区低成本乡村数字课堂 | M5+M7可部分覆盖，但"成本控制节点"缺失 |
| Q117 | 非洲NGO有限预算培训中心 | M8+M5可部分覆盖，但人道主义采购约束完全不同 |
| Q139 | 难民营儿童学习中心（快速低成本） | M8覆盖极端；"可扩展、可复制的低成本模块"逻辑缺失 |
| Q154 | 农村自建房有限预算质感提升 | M5偏在地文化；此题核心是预算分配策略 |
| Q167 | 四线城市高品质保障房 | M2+M4交叉，无法清晰指引"有限预算内的可感知升级" |
| Q171 | 有限财政文化综合体（三线城市） | M6偏区域更新；此题是"用设计杠杆替代资金" |

**这类问题的独特设计逻辑：**
- **价值杠杆思维**：识别哪3个节点高投入高回报，哪3个节点可极致压缩
- **视觉比例分配**：入口/核心视觉区满配，背景区域低配
- **材料替代系统**：高质感替代方案库（如仿石砖替代天然石）
- **施工策略控制**：预制化、分阶段、装配逻辑
- **长期维护成本入模型**：不只看建设成本，算5年总拥有成本

### 2.3 【结构性空白 C】临时/装置型设计 — 约 5 道题，逻辑完全不同

**识别信号**：空间存在的时间轴是**有限且预设的**，拆除是设计目标的一部分。

| 题号 | 问题核心 |
|------|----------|
| Q56 | 3个月户外音乐节VIP区（可快速搭建+拆卸+环保回收） |
| Q84 | 30小时奢侈品店换装（不闭店，预制装配） |
| Q108 | 可快速调整为派对/拍摄/居住的流动空间 |
| Q144 | 临时城市展馆（3个月后全部回收材料） |
| Q155 | 快闪品牌模块 + 可变展示橱窗系统 |

**这类问题的独特设计逻辑：**
- **可逆性是第一原则**：所有结构、材料、连接件都必须可无损拆卸
- **安装时间轴控制**：设计即施工流程设计
- **循环材料体系**：不只"可回收"，还要有回收路径
- **模块化运输约束**：空间单元必须适配物流尺寸
- **"消失"作为设计美学**：与M8（生存优先）的"永久性"完全相反

## 四、建议方案

### 方案A：新增 1 个模式（推荐）

**新增 M11｜健康疗愈型设计（Healthcare & Healing Design Mode）**

| 维度 | 说明 |
|------|------|
| 核心命题 | 空间作为疗愈工具——不是"好看"或"好用"，而是"让人安全地恢复" |
| 适用场景 | 医疗空间、养老社区、心理干预、无障碍住宅、康复训练、正念空间、临终病房、月子中心 |
| 核心决策原则 | **安全感优先**：每一个设计决策先问"会不会造成焦虑或伤害"，再问美学 |
| 与 M3（情绪体验）的本质区别 | M3追求高潮体验（让人愿意留下），M11是降低激活（让人安全放松） |
| 与 M9（社会结构）的本质区别 | M9处理权力关系，M11处理脆弱状态中的人的需求 |
| 能力核心 | A5灯光节律（降低色温、减少蓝光）、A9社会建模（无声关怀）、A4材料（低VOC、触感安全）、A6功能效率（动线减少碰撞）|
| 评估权重重构 | 生理舒适度(35%) + 心理安全(30%) + 功能可达性(20%) + 材料安全(15%) |

四个二级模块建议：
- **M11-1 心理安全环境系统**（色彩、声音、边界、隐私分级）
- **M11-2 生理疗愈节律控制**（光节律、温湿度、空气质量）
- **M11-3 无障碍尊严设计**（不暴露功能需求的无障碍方案）
- **M11-4 双视角服务动线**（服务提供者 × 被服务者动线分离）

### 方案B：强化 M4 增加子模块（强烈建议）

**M4-5 预算约束策略（Budget Constraint Intelligence）**

```
现有 M4 四个子模块：
  M4-1 客群资产模型
  M4-2 坪效转化模型
  M4-3 溢价符号构建
  M4-4 生命周期收益

新增：
  M4-5 预算约束策略（BCS）
     核心命题：在成本硬约束下，识别价值杠杆节点

     五步操作模型：
     1. 视觉价值扫描    —— 找出用户"第一视线"落点（决定全局感知）
     2. 杠杆节点识别    —— 通常 20% 的区域决定 80% 的价值感知
     3. 三重投入节点    —— 必须重金：入口/核心视觉区/材料节点
     4. 三重极简节点    —— 可极致压缩：背景区/储藏区/辅助区
     5. 长期成本入模    —— 维护周期 × 替换成本 × 耐久度
```

**触发条件**：当 `project_classification.budget_tier = "constrained"` 或关键词出现"预算有限/低成本/有限财政/万元改造/控制成本/压缩造价"时，自动激活 M4-5。

### 方案C：M8 增加人道主义子模块（建议）

**M8-5 人道主义应急设计（Humanitarian Emergency Design）**

适用：Q117（非洲NGO）、Q133（战后重建）、Q139（难民营）

```
与 M8 其他子模块的区别：
  M8-1/2/3/4 = "我们在极端环境中优化生活质量"
  M8-5 = "在资源极度匮乏环境中，用最少的材料提供最基本的尊严"

核心特征：
  · 采购约束（必须使用当地材料或特定援助物资）
  · 可复制性（设计必须能被非专业工人按模板执行）
  · 快速性（通常2-4周内落地）
  · 长期扩展能力（初始设计可随资源增加而升级）
  · 创伤敏感语言（战后/灾后空间语言必须消除威胁感）
```

### 方案D：临时装置型纳入 M11 运营产品化（or 独立）

**两种路径：**
- **路径1（轻量）**：在 M11（运营产品化）增加 M11-5"时间轴产品化（Time-based Productization）"，处理临时/快闪/循环场景
- **路径2（独立）**：新增 M12 临时装置型设计（Ephemeral Space Mode）

**建议路径1**，因为临时空间的核心逻辑是"产品模块化 × 时间轴可控"，是 M11 的特殊子集，数量（5题）也不足以支撑独立模式。

## 六、模式扩展后的新体系（建议版）

| 编号 | 模式 | 状态 | 覆盖新增的问题群 |
