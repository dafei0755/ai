# 行为设计框架 (Behavioral Design)

> 适用场景：用空间系统性影响人的决策和行为的分析框架

## 框架概述

行为设计框架将空间视为**行为干预工具**。它借鉴行为经济学的"助推"理论，通过环境
线索（而非强制规则）引导使用者的注意力流、移动路径和消费决策。核心理念：
**改变环境比改变人更有效**。

## 五大核心维度

### 1. 行为目标定义 (Behavior Target Definition)
- **关键问题**：希望空间促进/抑制哪些具体行为？
- **判断标准**：行为目标是否可观察、可测量（停留时间、购买转化率、路径选择比例）
- **常见误区**：目标模糊——"让顾客感到舒适"不是行为目标，"使顾客平均停留时间≥15分钟"是

### 2. 空间助推机制 (Spatial Nudge Mechanism)
- **关键问题**：哪些空间要素可以作为"助推"工具？
- **判断标准**：助推机制是否符合 Thaler 的"自由家长主义"——引导但不强制
- **常见误区**：将"助推"等同于"操控"——好的空间助推应保留使用者的选择自由

### 3. 注意力流设计 (Attention Flow Design)
- **关键问题**：使用者的视觉和注意力如何被空间序列引导？
- **判断标准**：是否绘制了"注意力地图"——从入口到核心区域的视觉引导路径
- **常见误区**：过度激烈的注意力争夺——每面墙都在"喊叫"

### 4. 转化路径 (Conversion Pathway)
- **关键问题**：从兴趣→行动→结果的空间化转化漏斗是如何设计的？
- **判断标准**：是否识别了转化漏斗中的"泄漏点"并设计了空间化解决方案
- **常见误区**：只关注入口吸引力，忽略深层转化（如收银区的排队焦虑导致放弃购买）

### 5. A/B实验协议 (A/B Experiment Protocol)
- **关键问题**：行为设计的效果如何通过对照实验验证？
- **判断标准**：是否预设了可实施的A/B测试方案——控制变量、样本量、测量指标
- **常见误区**：假设空间设计天然有效，不设置效果验证环节

## 分析深度指南

| 层级 | 内容要求 |
|------|----------|
| L1 描述 | 识别目标行为、用户群体、空间触点 |
| L2 分析 | 绘制当前行为动线 + 注意力热点图 |
| L3 策略 | 设计空间助推方案和转化路径 |
| L4 验证 | 设计 A/B 测试方案 + 数据采集计划 |
| L5 远见 | 长期行为数据驱动的空间迭代机制 |

## 知识注入要点

- Nudge Theory：Thaler & Sunstein《助推》——选择架构与默认选项
- Paco Underhill《顾客为什么购买》——零售空间中的行为观察方法论
- 注意力经济学：稀缺注意力资源在空间中的分配
- 热力图分析：红外追踪、WiFi探针、视线追踪技术的应用
- 中国语境：商业综合体的坪效优化实践、新零售空间体验设计
