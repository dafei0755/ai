# 竞标策略框架 (Competitive Bidding)

> 适用场景：设计竞标的竞争分析、差异化定位与提案策略

## 框架概述

竞标策略框架将设计竞赛/投标从"做一个好方案"提升为**"做一个赢的方案"**。
它不替代设计能力，而是**在同等设计水平下，通过策略性思维提升中标概率**。

## 五大核心维度

### 1. 竞争对手分析 (Competitor Analysis)
- **关键问题**：其他竞标团队可能的设计方向和策略是什么？
- **判断标准**：是否研究了竞争对手的历史作品、团队构成和设计倾向
- **常见误区**：只关注自己的方案，对竞争环境缺乏感知

### 2. 差异化策略 (Differentiation Strategy)
- **关键问题**：如何在评委心中建立独特且难以复制的定位？
- **判断标准**：差异化是否基于真实能力而非噱头——"我们能做到而别人做不到的是什么"
- **常见误区**：为差异化而差异化——提出哗众取宠但不可落地的方案

### 3. 提案叙事 (Proposal Narrative)
- **关键问题**：提案的叙事结构如何引导评委从"看方案"到"认同方案"？
- **判断标准**：提案是否有清晰的"问题→洞察→策略→方案→愿景"叙事弧线
- **常见误区**：提案沦为技术图纸集合，缺乏打动人心的叙事

### 4. 风险对冲 (Risk Hedging)
- **关键问题**：方案中哪些部分可能被评委挑战？如何预设回应策略？
- **判断标准**：是否做了"魔鬼辩护"——自己先攻击自己的方案弱点
- **常见误区**：只展示方案优势，对弱点毫无准备

### 5. 客户心理 (Client Psychology)
- **关键问题**：甲方/评委的真实诉求（而非招标文件中的表面需求）是什么？
- **判断标准**：是否研究了甲方的组织文化、决策风格、历史偏好
- **常见误区**：将招标文件当作唯一的需求来源

## 分析深度指南

| 层级 | 内容要求 |
|------|----------|
| L1 描述 | 竞标规则梳理、评委构成分析 |
| L2 分析 | 竞争对手画像 + 甲方心理分析 |
| L3 策略 | 差异化定位 + 提案叙事结构 |
| L4 验证 | 魔鬼辩护 + 弱点对冲方案 |
| L5 远见 | 中标后的持续合作策略 |

## 知识注入要点

- 竞标策略：建筑竞赛的策略性思维（非纯设计思维）
- 提案叙事：McKinsey SCQA (Situation-Complication-Question-Answer) 框架
- 评委心理：锚定效应、峰终定律在提案展示中的应用
- 视觉策略：信息设计在竞标文本中的作用——图表优于文字
- 中国语境：国内设计竞标的潜规则、招投标法规、评审流程特点
