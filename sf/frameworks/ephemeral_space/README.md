# 临时/事件空间设计框架 (Ephemeral Space)

> 适用场景：生命周期 3 天 - 3 个月的短周期空间设计范式

## 框架概述

临时空间设计框架处理**有限生命周期内的最大体验密度**问题。与永久性建筑不同，
临时空间的设计逻辑颠覆了耐久性优先的原则，转而追求**快速搭建、高密度体验、
完整拆卸、材料回收**的全周期管理。

## 五大核心维度

### 1. 生命周期规划 (Lifecycle Planning)
- **关键问题**：从设计到搭建到运营到拆除到回收，每个阶段的时间预算？
- **判断标准**：是否提供了精确到天的全周期甘特图，标注关键里程碑
- **常见误区**：只设计"展览期"，忽略搭建和拆除阶段的空间/时间约束

### 2. 快速装配逻辑 (Rapid Assembly Logic)
- **关键问题**：结构体系如何支持快速搭建（目标：小时级或天级）？
- **判断标准**：是否采用模块化/可重复使用的结构系统，搭建不依赖专业工种
- **常见误区**：用永久性建筑思维设计临时空间——湿作业、现浇混凝土

### 3. 材料循环性 (Material Circularity)
- **关键问题**：材料在事件结束后的去向——回收、再利用、升级再造还是填埋？
- **判断标准**：是否计算了材料循环率，提出了具体的材料去向方案
- **常见误区**：使用一次性材料后全部废弃，或使用"可回收材料"但没有实际回收渠道

### 4. 品牌表达密度 (Brand Expression Density)
- **关键问题**：在有限空间和时间内，如何最大化品牌/主题的表达强度？
- **判断标准**：每平方米是否承载了足够的品牌信息且不显杂乱
- **常见误区**：信息过载或过于空旷——未找到"密度甜蜜点"

### 5. 事后策略 (Post-Event Strategy)
- **关键问题**：临时空间结束后的遗产是什么——数字资产、社交传播、场地恢复？
- **判断标准**：是否设计了"空间消失后的持续存在"策略（如社交媒体打卡点、NFT纪念）
- **常见误区**：事件结束=项目结束，不考虑长尾效应

## 分析深度指南

| 层级 | 内容要求 |
|------|----------|
| L1 描述 | 明确生命周期、场地条件、参观人流预估 |
| L2 分析 | 搭建-运营-拆除的可行性评估 |
| L3 策略 | 模块化设计方案 + 材料循环方案 |
| L4 验证 | 搭建时间测试 + 承载力验证 |
| L5 远见 | 事后遗产策略 + 模块库复用规划 |

## 知识注入要点

- 临时建筑理论：Shigeru Ban 纸管建筑、MVRDV 临时性策略
- 展览设计：Atelier Brückner 的叙事空间方法论
- 循环经济：Cradle to Cradle（摇篮到摇篮）理念在临时空间中的应用
- 活动管理：Festival/Event 空间的人流动力学模拟
- 中国语境：故宫/国博特展设计、品牌快闪店（Pop-up Store）的中国实践
