# 医疗/治愈空间专项框架 (Healthcare & Healing)

> 适用场景：医疗级空间规范与治愈性设计的交叉框架

## 框架概述

医疗/治愈空间框架处理两个常被割裂的需求维度：**临床合规性**（感染控制、设备布局、
应急动线）和**治愈性环境**（自然光引入、减压色彩、噪声控制）。框架的价值在于让
两者不是互相妥协，而是互相增强。

## 五大核心维度

### 1. 临床合规 (Clinical Compliance)
- **关键问题**：空间是否满足医疗级别的硬性规范（洁净度、感控分区、无障碍）？
- **判断标准**：是否引用了具体的国标/行标（如 GB 51039 综合医院建筑设计规范）
- **常见误区**：过度关注治愈性而忽略基础合规，或反之

### 2. 治愈氛围 (Healing Atmosphere)
- **关键问题**：环境因素（光线、色彩、声学、气味、绿植）如何系统化设计以促进康复？
- **判断标准**：是否基于循证设计研究，而非主观判断"绿色让人放松"
- **常见误区**：将治愈性等同于"温馨装饰"，缺乏系统化的环境干预策略

### 3. 心理通道 (Psychological Pathway)
- **关键问题**：患者/家属/医护人员的心理旅程如何被空间支持？
- **判断标准**：是否绘制了"情绪地图"——从入口焦虑到诊疗信任到康复希望的空间序列
- **常见误区**：只关注患者体验，忽略医护人员的精神健康空间需求

### 4. 安全协议 (Safety Protocol)
- **关键问题**：紧急情况（火灾、暴力事件、院感爆发）的空间应对方案？
- **判断标准**：是否设计了多层级安全响应——日常预防、预警响应、紧急疏散、灾后恢复
- **常见误区**：安全设计仅停留在消防规范层面

### 5. 照护工作流 (Caregiver Workflow)
- **关键问题**：医护人员的工作流线是否高效且不增加患者干扰？
- **判断标准**：是否分析了护理站-病房-检查室-药房之间的距离和频率矩阵
- **常见误区**：空间美观性优先于工作流效率

## 分析深度指南

| 层级 | 内容要求 |
|------|----------|
| L1 描述 | 识别医疗类型、患者群体、合规等级 |
| L2 分析 | 临床合规审查 + 循证治愈要素评估 |
| L3 策略 | 合规与治愈的融合设计方案 |
| L4 验证 | POE评估方案 + 患者满意度测量设计 |
| L5 远见 | 适应未来医疗模式变化的弹性设计 |

## 知识注入要点

- 循证设计：Roger Ulrich "View Through a Window" 开创性研究
- Planetree 模式：以患者为中心的医疗环境设计理念
- Maggie's Centres：癌症关怀中心的建筑治愈哲学
- 中国语境：中医"天人合一"理念与现代治愈性设计的结合
- 感控规范：GB 51039、JCI标准中的空间设计要求
