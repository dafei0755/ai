# 人道主义设计框架 (Humanitarian Design)

> 适用场景：极端社会/政治约束下的设计策略——难民营、灾后重建、紧急援助空间

## 框架概述

人道主义设计框架处理**极端约束条件**下的设计问题。它与常规项目框架的根本区别在于：
使用者群体可能经历了创伤，供应链可能断裂，文化背景高度敏感，且"足够好"远比
"完美"更重要——**速度和尊严是设计的首要优先级**。

## 五大核心维度

### 1. 约束地图 (Constraint Mapping)
- **关键问题**：物资、人力、时间、政治、安全的具体约束条件？
- **判断标准**：是否绘制了多维约束地图，标注硬约束（不可突破）和软约束（可协商）
- **常见误区**：用常规项目思维评估约束——"这里没有混凝土搅拌站"不是问题，而是设计条件

### 2. 创伤敏感设计 (Trauma-Sensitive Design)
- **关键问题**：空间如何避免触发创伤记忆，同时提供安全感和控制感？
- **判断标准**：是否咨询了心理学专业人员，识别了特定创伤类型的空间触发物
- **常见误区**：将"安全感"等同于"围合感"——对经历过监禁的人群，过度围合可能是创伤触发

### 3. 物流链适应 (Logistics Chain Adaptation)
- **关键问题**：设计方案是否适配当地可获得的材料和运输条件？
- **判断标准**：材料清单是否基于当地市场调研而非标准化采购平台
- **常见误区**：设计方案依赖进口材料，忽略关税、运输时间和损耗率

### 4. 文化敏感性 (Cultural Sensitivity)
- **关键问题**：设计是否尊重使用者的文化习俗、宗教需求、性别规范？
- **判断标准**：是否与社区代表和文化顾问进行了充分沟通
- **常见误区**：以"普世人道主义"名义忽略具体文化差异——如在穆斯林社区不设性别分区

### 5. 可复制性模型 (Replicability Model)
- **关键问题**：设计方案是否可以在不同地点快速复制和本地化适配？
- **判断标准**：是否提供了"设计工具包"——当地人可以自行搭建/维护的简明指南
- **常见误区**：设计依赖专业施工队伍，无法在紧急情况下由非专业人员实施

## 分析深度指南

| 层级 | 内容要求 |
|------|----------|
| L1 描述 | 灾情/冲突概况、受影响人群画像、现有资源盘点 |
| L2 分析 | 约束地图绘制 + 创伤类型识别 |
| L3 策略 | 快速部署方案 + 本地化材料策略 |
| L4 验证 | 社区参与式验证 + 文化适宜性评审 |
| L5 远见 | 从应急到过渡到永久的渐进式升级路径 |

## 知识注入要点

- Sphere Standards：人道主义最低标准（住所/定居点）
- UNHCR 难民营规划指南：空间标准和布局原则
- Shigeru Ban：纸管建筑在灾后重建中的应用
- Architecture for Humanity / MASS Design Group：社区参与式设计实践
- 文化敏感设计：Aga Khan Award for Architecture 的评价维度
- 中国语境：汶川/雅安地震灾后重建经验、对口援建模式
